1.  In the provided codebox, enter the assembly program you want to evaluate with. You can use the provided examples by clicking on the 'Copy Example' button.
2.  Click 'Generate Trace' to generate the execution trace for the program. 
3.  Select the type of predictor you want to use (One-Bit or Two-Bit).
4.  Click on the 'Run Simulation' button to see the predictor's performance. The simulation will show the initial state, the prediction for each branch, the new state after the branch, and the overall accuracy of the predictor.
5.  You can reset the simulation at any time by clicking the 'Reset' button.
